HTML Table

Element Element Tag Function
Hyper Text Markup Language html

Encapsulates the whole html code. It is the first and most important element in a html document. It is considered the root element of the html document.

Head head

Contains important elements of the html document such as stylesheet references, the title of the website, and the characterset that the html file uses.

Body body

The element tag that encapsulates most of the visible content of the page. It contains elements such as img, p, h1, or similar ones that typically are content elements.

Heading h1

The header element gives text higher weight and important when viewing a website. It bolds and enlargens text, giving organization to your website content.

Paragraph p

The p element usually contains the most text. It is most commonly used for long text based content on your website.

CSS Table

Property Description
Colors

Different Color Properties

color:

The color: CSS selector sets the color for foreground elements. In text elements it modifies text color.

background-color:

The background-color: CSS selector sets the color for background elements. It is most useful for setting the design of the website as it can fill spaces with color.

Box Model

Components of the Box Model:

margin:

The margin is the space around an element. It enables you to space elements from each other. It is very useful for creating some necessary whitespace between elements.

border:

The border selector enables you to set the border width, border style, and the border color. An example would be:

border: 2px solid red

This would create a 2px wide and solid red border around an element.

padding:

Padding allows you to make your element take space. If you want an element such as a button to be bigger, add padding. Padding is useful when designing how your elements in your website take up space.

Content

Content refers to the actual content of your element. It may be text or media like an image.

Display

Different Display Techniques

Inline

Displays element as an inline element.

Block

Displays element as a block element.

Flex

Puts elements into a flex container. Has properties such as flex-direction and align-items.

Grid

Put elements into a grid container. Can set column and row properties for elements within the container.
